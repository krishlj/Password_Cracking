This project provides hands-on experience in cryptography, password hashing, hash identification, and authorized password auditing. Using Hashcat, John the Ripper, and CrackStation, you will identify and test common hash types, including MD5, SHA, NTLM, and Linux password hashes.

The project also demonstrates file-integrity verification and password recovery from protected ZIP files, RAR archives, and SSH private keys. Each task includes practical training files and commands for independent implementation. By completing this project, you will understand why weak passwords and outdated hashing algorithms create security risks and learn defensive recommendations for stronger password storage, authentication, and credential protection.


Note: Download Hashing-Basics.zip for a practical demonstration.
